# Frangipani 一种可扩展的分布式文件系统

## 0 概要
- coherent / shared access 连贯的共享访问
- arbitratily scalable 随意扩展的
- 最少的认为干预，管理工作不能因为组件的加入而复杂
- 两层架构，底层是Petal虚拟硬盘，高层是Frangipani提供分布式锁服务
- 在安全协议下，节点之间可以相互信任
- 非安全的机器，则可以通过普通的网络协议共享
